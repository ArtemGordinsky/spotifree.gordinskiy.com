Troubleshooting

Spotifree gives you a headache?

Follow these steps to fix Spotifree:

  1. Kill Spotifree process (if any) using the Activity Monitor (Applications → Utilities → Activity Monitor);

  2. Get a fresh app from the homepage. Install it, replacing the old one.

  3. Run the app again.

  4. If your problem is still not fixed, send me a message.
