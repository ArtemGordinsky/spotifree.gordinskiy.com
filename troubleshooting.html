Troubleshooting

Let's try and make it work

Spotifree doesn't work? Try these:

  1. Reset Spotify (don't worry, all your playlists and saved tracks will be there when you're done)

    1. Quit Spotify;
    2. Open Finder then click Go > Library in the menu bar. (You may need to hold the Alt key if Library isn’t visible);
    3. Open Caches and delete the com.spotify.Client folder;
    4. Click the back arrow;
    5. Open Application Support and delete the Spotify folder ;
    6. Reopen Spotify, sign in with Facebook or your login and password.

  2. Reinstall Spotifree

    1. Terminate Spotifree process (if any) using the Activity Monitor (Applications > Utilities > Activity Monitor);
    2. Get a fresh app from the homepage. Install it, replacing the old one.
    3. Open Spotifree app again.

  3. Use an older version of Spotify (if none of the previous options helped)

    1. Quit Spotify;
    2. Go to this website;
    3. Download version 0.9.15.17;
    4. Replace your Spotify app with the downloaded one;
    5. Press ⇧ + ⌘ + G in Finder, paste in ~/Library/Application Support/Spotify;
    6. If present, delete Spotify_new.archive and Spotify_new.archive.sig;
    7. Open a text editor;
    8. Create two blank documents named Spotify_new.archive and Spotify_new.archive.sig (be sure that extension is not hidden);
    9. Save both to the previously opened folder;
    10. Right click each file and go to Get Info
    11. Check the Locked checkbox on each and close the windows;
    12. Quit and open Spotifree again;
    13. Open Spotify app.

If you have tried all of the options above and the problem is still there, feel free to shoot me a message.
